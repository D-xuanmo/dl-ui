# 介绍

> 一个基于 `Vue3` 开发的一个组件库

### dl 释义
- `d` 代表梦想（Dream）、动态（Dynamic）
- `l` 代表低代码（Low code）

## 亮点

- 🚀 用最少的代码，解决表单渲染，主打 `JSON` 配置模式，低代码方向；
- 🚀 零外部依赖，不依赖其他三方包；
- 💪 内置常用表单组件，支持扩展集成到表单使用；
- 💪 繁琐的表单校验通过规则配置即可，校验与表单业务可完美剥离；
- ✍️ 使用 TypeScript 开发，提供完整的类型定义；
- 🛠 更多功能开发中，欢迎共建...
